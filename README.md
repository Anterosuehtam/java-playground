# Java Playground

Repositório dedicado à prática e consolidação dos fundamentos de Java.

## 📚 Conteúdos abordados

- Estruturas de dados (Collections)
- Programação orientada a objetos (POO)
- Streams e programação funcional
- Tratamento de exceções
- Consumo de APIs
- Fundamentos da linguagem

## 📂 Estrutura

- collections → List, Map, Set
- streams → filter, map, reduce, sorted
- poo → classes, herança, polimorfismo, interfaces
- fundamentals → estruturas básicas da linguagem
- exceptions → tratamento de erros
- api → consumo de APIs externas e internas

## 🎯 Objetivo

Consolidar a base em Java e evoluir para desenvolvimento de aplicações mais complexas.

## 🚀 Observações

Os códigos aqui são exercícios práticos focados em aprendizado, organização e evolução contínua.
